## Prerequisites

- python==3.9.10

## How to start

```shell
$ git clone {username/repo addr}
$ cd {repo addr}

$ python fizzbuzz.py
```

## Installation

```shell
$ poetry install

or

$ pip install -r requirements.txt
```

- Print fizzbuzz
